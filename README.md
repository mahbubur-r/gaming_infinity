- **Navbar Section:**

  A stunning image slider showcasing featured games with autoplay and dynamic transitions powered by **Swiper**.

  Optimized layouts for mobile, tablet, and desktop devices with seamless UI responsiveness.

  A clean, modern newsletter section that connects to **Formspree**, letting users subscribe for updates effortlessly.

  Crafted using **Tailwind CSS** and **DaisyUI**, ensuring a beautiful, futuristic, and consistent design.  

- **Icons:**  
  Enhanced visuals using **React Icons** for interactive UI elements and buttons.
